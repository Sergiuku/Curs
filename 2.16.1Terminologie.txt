2.1  - Metalimbaj - metalanguage - informatii prin care se descrie un limbaj.
2.2  - Echo, este o comanda pentru output ce primeste parametrii.
2.3  - Variabila(variable) este un simbol sau un nume ce tine loc pentru o valoare, 
	retine o valoare in memorie, o cantitate de informatie ce poate fi reutilizata in codul viitor.
     - Int, este un intreg, contine toate numere intregi, pozitive si negative ce pot fi date unei variabile.
2.4  - Alogrithm, un set de instructiuni, modele prin care atingi un anumit task.
2.5  - Conjunctie(and), este o operatie booleaana care va fi adavarata numai atunci cand ambele expresii booleene A si B vor fi adevarate.
       Expresiile boolene A si B pot fi variable, operatii sau o conditie.
2.6  - If este un construct urmat de o expresie logica ca si conditie. In functie valoarea de adevar a conditiei se va executa ramura if sau 
     - ramura Else. Ramura if se va executa atunci cand conditia este adevarata, iar ramura else atunci cand valoarea de adevar a ramurei if este falsa.
2.7  - While Loop, contine o conditie si un bloc de instructiuni. Cu ajutorul while loop-ului putem executa blocul de instructiuni atat timp cat conditia este adevarata.
2.8. - Foreach Loop, funtioneaza cu ajutorul unui array. Se foloseste atunci cand avem nevoie de afisarea valorilor unui array intr-o bucla.
2.9  - For loop, este o bucla ce ruleaza un set de instructiuni si este folosita atunci cand cunosti dinainte de cate ori trebuie sa ruleze bucla.
2.10 - Constanta(constants), se comporta la fel ca si o variabila numai ca diferenta intre ele este ca constanta nu isi va putea schimba valoarea o data ce a fost definita.
     - String double-quoted, acestea interpreteaza variabilele din interiorul ghilimelelor si le transforma in valorile variabilelor.
2.11 - $_GET, este o variabila ce permite utilizatorului sa introduce anumite valori prin intermediul URL-ului.
2.12 - Multidimensional Array, un array ce poate contine mai multe array-uri.
2.13 - Binary or, operatie ce verifica daca bitii dintre 2 valori sunt setati.
2.14 - Variabila Variabila, este o variabila construita pe baza valorii altei variabile.
